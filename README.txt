Un jeu de coinche client / serveur

Ce dossier contient 2 dossiers principaux :

- le dossier client qui contient les sources pour compiler la partie cliente du projet. Une fois compilées avec Visual Studio, le binaire pourra être retrouvé dans DOT_cardGames_2017/client/bin/Debug, et ce lance de la manière suivante :
./client.exe IP_SERVEUR PORT

- le dossier serveur qui contient les sources pour compiler la partie serveur du projet. Une fois compilées avec Visual Studio, le binaire pourra être retrouvé dans DOT_cardGames_2017/serveur/bin/Debug, et ce lance de la manière suivante :
./serveur.exe PORT

Le jeu débutera lorsque 4 joueurs seront connectés en même temps. Un salon sera alors créé pour les 4 joueurs et les instructions seront données.
